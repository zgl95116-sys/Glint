# Glint Harness — Blocked Features

> When an agent gets stuck on a feature after 2+ attempts, it documents the blocker
> here and picks a different feature next round. A human (or a future agent with
> new context) can unblock later by deleting the entry.

## Format

```
### {FEATURE_ID}

**Blocked since:** {ISO_TIMESTAMP}
**Attempts:** N
**Symptom:** What goes wrong each time
**Root cause hypothesis:** Why you think it fails
**What's needed to unblock:** Specific info/decision/tool required
**What NOT to try again:** Approaches already ruled out
```
